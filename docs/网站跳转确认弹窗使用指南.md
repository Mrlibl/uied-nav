# 网站跳转确认弹窗使用指南

## 📋 概述

本指南介绍如何在UIED设计导航项目中使用网站跳转确认弹窗功能。该功能参考了[aibase免责声明页面](https://top.aibase.com/excesslink?py=infatuated-ai)的设计理念，为用户提供安全的外部链接访问体验。

## 🎯 功能特性

### ✅ 核心功能
- **安全警告** - "即将离开UIED设计导航"提示
- **目标网站信息** - 显示网站名称、URL和描述
- **安全检测** - 5秒倒计时模拟安全检测过程
- **风险提示** - 提醒用户保护账号和隐私信息
- **免责声明** - 参考aibase的法律免责条款

- **推荐资源** - 展示相关设计资源Banner
- **响应式设计** - 完美支持移动端和桌面端
- **无障碍支持** - 符合WCAG标准的键盘导航

## 📖 使用方法

### 1. 基础用法

```tsx
import React from 'react';
import { WebsiteExitModal } from '../components/UI';
import { useWebsiteExit } from '../hooks/useWebsiteExit';

const MyComponent: React.FC = () => {
  const {
    isModalVisible,
    currentWebsite,
    hideExitModal,
    confirmVisit,
    handleExternalLinkClick,
    reportWebsite
  } = useWebsiteExit();

  const website = {
    name: 'UIED设计灵感',
    url: 'https://www.uied.cn/category/wenzhang/ganhuo',
    description: 'UIED技术团队精选的设计干货文章'
  };

  return (
    <div>
      <a
        href={website.url}
        onClick={(e) => handleExternalLinkClick(e, website)}
      >
        访问UIED设计灵感
      </a>

      <WebsiteExitModal
        visible={isModalVisible}
        website={currentWebsite}
        onClose={hideExitModal}
        onConfirm={confirmVisit}
        onReport={reportWebsite}
      />
    </div>
  );
};
```

### 2. Hook API 详解

#### `useWebsiteExit()`

**返回值：**
```typescript
{
  // 状态
  isModalVisible: boolean;        // 弹窗显示状态
  currentWebsite: WebsiteInfo | null;  // 当前目标网站信息
  
  // 操作方法
  showExitModal: (website: WebsiteInfo) => void;        // 显示弹窗
  hideExitModal: () => void;                            // 隐藏弹窗
  confirmVisit: () => void;                             // 确认访问
  handleExternalLinkClick: (e: Event, website) => void; // 处理链接点击
  reportWebsite: (url: string, reason: string) => void; // 举报网站
}
```

#### `WebsiteInfo` 接口
```typescript
interface WebsiteInfo {
  name: string;          // 网站名称
  url: string;           // 网站URL
  description?: string;  // 网站描述（可选）
}
```

### 3. 在工具卡片中使用

```tsx
// ToolCard组件示例
const ToolCard: React.FC<{ tool: Tool }> = ({ tool }) => {
  const { handleExternalLinkClick } = useWebsiteExit();

  return (
    <div className="tool-card">
      <h3>{tool.name}</h3>
      <p>{tool.description}</p>
      <a
        href={tool.url}
        onClick={(e) => handleExternalLinkClick(e, {
          name: tool.name,
          url: tool.url,
          description: tool.description
        })}
        className="tool-link"
      >
        访问工具 ↗
      </a>
    </div>
  );
};
```

### 4. 批量链接处理

```tsx
const LinkList: React.FC = () => {
  const { handleExternalLinkClick } = useWebsiteExit();

  const websites = [
    { name: 'Figma', url: 'https://figma.com', description: '界面设计工具' },
    { name: 'Sketch', url: 'https://sketch.com', description: 'Mac设计软件' }
  ];

  return (
    <div>
      {websites.map((website, index) => (
        <a
          key={index}
          href={website.url}
          onClick={(e) => handleExternalLinkClick(e, website)}
        >
          {website.name}
        </a>
      ))}
    </div>
  );
};
```
